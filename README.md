# Game_Plakhotnikov_TimeCoordination

Одностраничный модуль курсовой работы «Координация по времени». В проекте реализованы
три уровня игры с динамическими заданиями, мини-авторизацией, системой очков и таблицей рейтинга.

## Структура страниц
- `index.html` — заставка, ввод имени, выбор сложности и краткое описание уровней.
- `game.html` — игровая сцена с тремя уровнями (лампа + клавиатура, туннель + drag-n-drop, прыжки + двойной клик);
  на странице отображаются счет, штрафы, общий таймер и прогресс по раундам.
- `leaderboard.html` — финал и рейтинг: вывод статистики текущей попытки и списка лучших игроков.

## Основные требования, которые закрывает модуль
- Мини-авторизация и сохранение игрока между сессиями (LocalStorage + JSON).
- 3 уровня сложности, в каждом уровне генерируется 3–5 вопросов с псевдослучайными параметрами.
- Различные события: `keydown` (Space), перетаскивание `pointermove/pointerup`, двойной клик `dblclick`.
- Ограничение по времени на игру + ускорение в зависимости от выбранной сложности.
- Подсчет очков, штрафных баллов, динамическая сложность и проверка условий перехода между уровнями.
- Возможность завершить уровень/игру в любой момент и начать заново с новым именем.
- Анимации, случайное размещение игровых элементов, динамическое создание DOM-структур.
- Передача данных на страницу рейтинга и хранение истории для топ-20 результатов на устройстве пользователя.

## Запуск локально
1. Откройте папку `Game_Plakhotnikov_TimeCoordination` в IDE.
2. Дважды кликните `index.html` (или используйте локальный статический сервер, например `npx serve .`).
3. Следуйте шагам на экране: ввод имени → выбор сложности → прохождение уровней → просмотр рейтинга.
